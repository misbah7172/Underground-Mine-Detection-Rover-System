# PHASE 1 — Parts and Shopping List

- ESP32 DevKit V1 (x1)
- BTS7960 Motor Driver Module (x2)
- DC Geared Wheel Motors (matched pair)
- 12V Li-ion battery pack (appropriate capacity)
- Buck converter for 5V (for accessories)
- Emergency stop switch (NC)
- Connectors, terminal blocks, wire (16–22 AWG as needed)
- Heat-shrink, zip ties, mounting hardware
- Multimeter, small oscilloscope (recommended)
- Inline fuse and fuse holder
- 220µF electrolytic capacitor
- 0.1µF ceramic capacitor
- Logic-level jumper wires for control signals
- Prototype perfboard or terminal breakout board
